Pygame

I developed a 2D shooting game using Pygame, a popular game development library in python. The game features player movement, bullet shooting, enemy spawning, sound effects, and a life system. It's a fun, beginner-friendly arcade-style game that combines logic, amimation, and interactive design.

  • Player movement using arrows keys
  • Bullet firing machanics with collision detection
  • Enemies falling from the top of the screen.
